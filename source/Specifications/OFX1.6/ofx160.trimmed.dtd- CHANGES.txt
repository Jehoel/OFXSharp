	To:
	```
	<!-- ampersand -->
	<!ENTITY amp "&#38;">

	```

DTD Syntax Reminder:

* Visual Studio interprets *.dtd files as XML DTDs, not SGML DTDs, so it does not recognize the "- -" / "- O" / "O O" syntax for specifying start/end-tag options.

* `<!ELEMENT {elementName} - - {content}>` - Open and close tags required.
* `<!ELEMENT {elementName} - O {content}>` - Open tag required. Close tag optional.
* `<!ELEMENT {elementName} O O {content}>` - Opan and close tags optional. (where is this used?)

* Content syntax:

	* `()` - Defines a group.
	* ``
